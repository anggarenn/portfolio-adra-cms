# 🏢 Adra Chemical - CMS & Company Profile

*(Catatan: Source Code asli bersifat PRIVATE untuk melindungi privasi perusahaan. Ini adalah repository showcase untuk keperluan portfolio.)*

## 📖 Tentang Proyek Ini
Proyek ini adalah sistem website perusahaan (Company Profile) dinamis yang dilengkapi dengan Sistem Manajemen Konten (CMS) kustom untuk PT Adra Cipta Chemindo.

## 🚀 Fitur Utama
*   **Role-Based Access Control (RBAC):** Login aman dengan SSO Google dan pembagian role (Superadmin, Editor, User).
*   **Dynamic Module Manager:** Memungkinkan admin mengganti susunan layout dan konten secara instan.
*   **SEO Automation:** Sistem cerdas yang membuat URL bersih dan *auto-generate* Meta Description untuk mesin pencari Google.
*   **Inquiry System:** Sistem penawaran yang terintegrasi langsung dengan notifikasi Email (PHPMailer) dan kotak masuk (Inbox) di dashboard admin.
